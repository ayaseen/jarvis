📝 Quick Start Guide

vLLM vs Ollama: vLLM provides 2-3x faster inference with better GPU utilization. It loads one model at startup for maximum performance.

To change the model:

  1. Select a model from the list or enter a custom HuggingFace model path
  2. Adjust the context length and GPU memory settings if needed
  3. Click "Update Configuration"
  4. Restart the vLLM container:
docker-compose restart vllm llm-service

Monitor logs:

docker-compose logs -f vllm

Performance Tips:

  • AWQ models offer the best quality/speed ratio for RTX 4070 Ti
  • Set GPU memory to 90% for optimal performance
  • Reduce max_model_len if you encounter OOM errors
  • Expect 50-100+ tokens/sec with AWQ models
